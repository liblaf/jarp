# API Reference

Use these pages when you need the exact module surface rather than the
workflow guides.

- [`jarp`](jarp/README.md) covers the top-level re-exports.
- [`jarp.lax`](jarp/lax/README.md) documents control-flow wrappers with optional eager fallbacks.
- [`jarp.tree`](jarp/tree/README.md) covers PyTree partitioning, flattening, and registration helpers.
- [`jarp.tree.attrs`](jarp/tree/attrs/README.md) covers the `attrs` decorators and field specifiers.
- [`jarp.tree.codegen`](jarp/tree/codegen/README.md) covers the generated callback helpers behind custom registrations.
- [`jarp.tree.prelude`](jarp/tree/prelude/README.md) covers `Partial`, `PyTreeProxy`, and the built-in adapter registrations.
- [`jarp.warp`](jarp/warp/README.md) covers array conversion and Warp/JAX adapter wrappers.
- [`jarp.warp.types`](jarp/warp/types.md) covers precision-aware Warp dtype helpers.
